Report a bug

If you encounter a bug in IronPython, or would like to ask for a feature, please submit an issue on the IronPython GitHub project. Take a look through the existing issues to see if others are having the same problems you are. When creating an issue, keep in mind a bug report should include as much information as possible.

Ask the mailing list

If all else fails and you just want to talk to someone, or interested in tracking IronPython more closely, please subscribe to the IronPython users mailing list.

Please search the archives to see if someone has had the same questions as you.
